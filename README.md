## Overview

**Green Guardian** is a mobile application developed in **Android Studio (Java)** to support Israeli farmers impacted by the ongoing crisis. It connects farmers and consumers, providing a platform for direct communication and efficient agricultural management.

## Technologies

- **Java** | **Android Studio**
- **SQLite** | **API Integration**
- **AlarmManager** | **Fragments**

## Features

- Connects farmers with consumers
- Local data storage via SQLite
- Real-time notifications using AlarmManager
- Modular UI with Fragments

## Getting Started

1. Clone or download the repository.
2. Open the project in **Android Studio**.
3. Sync Gradle and run the app on an emulator or device.
